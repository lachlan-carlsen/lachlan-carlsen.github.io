MuASM

An Assembly Language for Music Nerds

Overview

MuASM (Music Assembly Language) is designed for manipulating musical data and synthesizing sound, with standardized assembly operations alongside unique, music production oriented instructions.

The benefit is making music creation even more math-based for people who enjoy visually looking at the math of things.

Key Features

  • Low-level control of sound synthesis
  • Music-oriented instructions like PLAYNOTE, TRANSPOSE, and SETKEY
  • Multiple waveform types (sine, square)
  • Register-based architecture for manipulating musical data
  • Support for key signatures and tempo control

Registers

Register Description
R0 General-purpose (default value: 0)
R1 Tempo control (beats per minute)
R2 Frequency control (in Hz)
R3 Note duration (in milliseconds)
R4 Volume level (0-100 scale)
R5 Sound wave type (0 = sine, 1 = square, etc.)
R6-R7 Reserved for sound effects
R8-R15 General-purpose registers

Basic Instructions

Name Syntax Functionality
ADD ADD R1, R2, R3 Adds R2 and R3; stores result in R1
SUB SUB R1, R2, R3 Subtracts R3 from R2; stores result in R1
MUL MUL R1, R2, R3 Multiplies R2 and R3; stores result in R1
MOV MOV R1, R2 Copies value from R2 to R1
LDI LDI R1, imm Loads an immediate value into R1

Music-Specific Instructions

Name Description
PLAYNOTE Plays a note with frequency in R2 and duration in R3
SETKEY Sets the global key signature to the immediate value (e.g., 0 = C Major, 1 = G Major)
TRANSPOSE Transposes the frequency in R2 up or down by the immediate value (in semitones)
TOGGLEWAV Changes the waveform type used for sound generation
DIS Sets a level of distortion for sounds within the function, on a scale of 0-10
STOPNOTE Stops the currently playing note

Sample Code

# Simple C major scale
main:
LDI R2, 261  # C4
LDI R3, 500  # 500ms duration
PLAYNOTE R2, R3

TRANSPOSE R2, 2  # D4
PLAYNOTE R2, R3

TRANSPOSE R2, 2  # E4
PLAYNOTE R2, R3

TRANSPOSE R2, 1  # F4
PLAYNOTE R2, R3

TRANSPOSE R2, 2  # G4
PLAYNOTE R2, R3

TRANSPOSE R2, 2  # A4
PLAYNOTE R2, R3

TRANSPOSE R2, 2  # B4
PLAYNOTE R2, R3

TRANSPOSE R2, 1  # C5
PLAYNOTE R2, R3
